# TestCraft AI-Assisted Automation Project
AI-assisted test automation project using TestCraft Chrome Extension


## Project Overview
This project demonstrates the use of TestCraft GPT-Powered Test Idea Generator & Automated Test Creator
(Chrome Extension v1.2.4) to analyze a live web application, generate test ideas using AI,
convert them into automated test flows, and validate core functionality efficiently.
